Ziel dieser Web-Seite

Der Zweck dieser Web-Seite ist, dass Sie die Möglichkeit haben, den HTML-Code näher kennen zu lernen.

Vergleichen Sie dazu immer wieder den Quellcode, den Sie hier in PhpStorm sehen, mit der Ausgabe in Ihrem Browser.

Was zuerst auffällt: «Tags!» –
All die vielen Abkürzungen in spitzen Klammern

HTML-Code besteht aus Tags und Text.
Etwas anderes als Tags und Text werden Sie im HTML-Code nie finden: Insbesondere finden Sie hier keine Bilder: Auf Bilder wird im Code nur verwiesen.

Das Allerwichtigste zuerst

  • Die Tags geben dem Browser an, welche «Rolle» ein Stück Text auf der Web-Seite spielt. Entsprechend muss (fast) jedes Element einer HTML-Seite in ein Tag-Paar eingeschlossen sein.
  • Tags sind (fast) immer paarweise angeordnet:
    So markiert z.B. der Tag <p> den Anfang eines Text-Absatzes, und der Tag </p> dessen Ende.

Ist ein Element unserer HTML-Seite...

  • ...eine Überschrift – z.B. der obersten Ebene? Dann muss es ins Tag-Paar
    <h1> ... </h1>
    eingebettet sein.
    Überschriften der zweitobersten Ebene kriegen das Tag-Paar <h2> ... </h2> und so weiter.
  • ...ein einfacher Text-Absatz? Dann muss er zwischen den Tags
    <p> und </p> stehen.
  • ...Teil einer Aufzählung? – Die dafür notwendigen Tags sehen Sie gleich hier im Quellcode zu dieser Stelle der HTML-Seite.
